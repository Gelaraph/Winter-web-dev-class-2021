CASCADING STYLE SHEET (CSS)

CSS (Cascading Style Sheets) is used for adding styles (e.g. fonts, colors, spacing) to Web documents.
The format for saving a CSS file is "the name .css"

e.g: style.css

To apply CSS to an HTML element you need to select the element and CSS provides you with a number of different ways to select the elements and apply rules to them.
But to understand how selectors work and their role in CSS, it's important to know the parts of a CSS rule.

A CSS rule is a block of code, containing one or more selectors and one or more declarations.


Anatomy of a CSS ruleset

A style sheet consists of a list of rules and each rule or rule-set consists of one or more selectors, and a declaration block.

Let's disect how CSS actually works using this diagram:


Explanation of the diagram

  • Selectors

    This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled (in this example, < p > elements). To style a different element, change the selector.

  • Properties

    In this example color is a property of the p element. In CSS, you choose which properties you want to affect to the element.

  • Property value

    To the right of the property—after the colon—there is the property value. This chooses one out of many possible appearances for a given property. (For example, there are many color values in addition to red.)

  • Declaration

    A declaration block includes a CSS property name and a property value pair which applies styles to the elements matched by the selectors. It should be noted however that a rule can contain more than one declaration and are separated by semicolons.
    Also, declaration blocks are surrounded by curly braces.
